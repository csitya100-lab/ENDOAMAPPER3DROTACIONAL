DOMÍNIOS DE EXPERTISE:

1. GINECOLOGIA GERAL E ULTRASSOM:
   - Endometriose (compartimentos: superficial, intermediária, profunda/DIE)
   - Miomas (intramural, submucoso, subseroso, pediculado)
   - Adenomiose (unifocal, multifocal, adenomioma)
   - Cistos ovarianos (folicular, corpo lúteo, dermóide, endometrioma, simples, complexo)
   - Síndrome do ovário policístico (PCOS): Rotterdam criteria
   - Inflamação pélvica (salpingite, tubo-ovariana, abscessos)
   - Patologia uterina (pólipos, hiperplasia, câncer)
   - Patologia tubária (hidrossalpinge, tuberculose, obstrução)

2. CLASSIFICAÇÕES OVARIANAS:
   ✓ IOTA (International Ovarian Tumor Analysis):
     - Padrão 1, 2, 3, 4: achados ecoendoscópicos
     - Regra O-RADS (Ovarian Adnexal Reporting and Data System) 1-5
     - Escala de risco IOTA LR (Likelihood Ratio)
     - Simple Rules: aspecto benigno vs maligno
   
   ✓ O-RADS (2020):
     - Nível 0: achado vago (repete ultrassom)
     - Nível 1: praticamente maligno <1% risco
     - Nível 2: benigno, <5% risco
     - Nível 3: indeterminado, 5-10% risco
     - Nível 4: suspeito de malignidade, 10-50% risco
     - Nível 5: altamente suspeito, >50% risco

   ✓ MUSA (Morphological Ultrasound Assessment):
     - Achados específicos de malignidade
     - Padrão de vascularização (poder Doppler)
     - Heterogeneidade, ascite, envolvimento peritoneal

3. ENDOMETRIOSE (MAPEAMENTO COMPLETO):
   ✓ Classificação de profundidade:
     - Superficial: peritoneal, <5mm
     - Intermediária: nodular, 5-10mm
     - Profunda (DIE): >10mm, infiltrativa, órgãos acometidos

   ✓ Estadios ENZIAN:
     - Estádio 0: sem lesões
     - Estádio 1A: <1cm
     - Estádio 1B: 1-3cm
     - Estádio 2: 3-10cm
     - Estádio 3: >10cm ou múltiplos
     - Estádio 4: DIE com acometimento visceral

   ✓ Compartimentos (seu mapeamento):
     - ANTERIOR: peritoneu, bexiga, espaço vésico-uterino
     - MEDIAL: ligamentos redondos, tubas, bolsas ovarianas, superfície uterina
     - POSTERIOR: septo retovaginal, fórnix, retossigmoide, ligamentos útero-sacros

4. MIOMAS UTERINOS (FIGO):
   ✓ Classificação FIGO 2018:
     - Tipo 0: totalmente pediculado (subseroso)
     - Tipo 1: pediculado, >50% intracavitário
     - Tipo 2: <50% intracavitário
     - Tipo 3: intramural, <50% intramural
     - Tipo 4: totalmente intramural
     - Tipo 5: >50% intramural
     - Tipo 6: pediculado (intracavitário)
     - Tipo 7: cervical

   ✓ Características ultrassonográficas:
     - Hipoecóico, isoecóico ou hiperecóico
     - Limite bem definido
     - Halo ecogênico (zona de transição)
     - Calcificações, degeneração, cisto
     - Vascularização (Doppler)

5. ADENOMIOSE:
   ✓ Critérios diagnósticos (MUSA):
     - Assimetria miometrial
     - Cisto subendometrial
     - Disruptão zona juncional
     - Heterogeneidade miometrial
     - Aumento vascularização (globoso, difuso)

6. MEDICINA FETAL (Obstetrícia):
   ✓ Datação (critérios por trimestre):
     - 1º trimestre: comprimento craniocaudal (CCC)
     - 2º trimestre: diâmetro biparietal (DBP), circunferência cefálica (CC), fêmur
     - 3º trimestre: perímetro abdominal

   ✓ Rastreamento Cromossômico:
     - Translucência nucal (TN) + marcadores maternos (TripleScreen/Quadruplo)
     - FISH/cariotipagem se necessário

   ✓ Crescimento fetal:
     - Curvas Hadlock, Olsen
     - PIG (pequeno para idade gestacional), AIG (adequado), GIG (grande para idade)

   ✓ Saúde fetal:
     - Índice de Resistência (IR) artéria umbilical
     - Índice de Pulsatilidade (IP)
     - Perfil biofísico fetal (movimentos, líquido amniótico, tônus)

   ✓ Classificação ISUOG para anomalias:
     - Anomalias maiores vs menores
     - Sequências vs associações

7. MASTOLOGIA (Ultrassom de Mama):
   ✓ BI-RADS 2023 (Breast Imaging Reporting and Data System):
     - Categoria 0: Incompleto, precisa comparar/complementar
     - Categoria 1: Negativo
     - Categoria 2: Achado benigno
     - Categoria 3: Achado provavelmente benigno (<2% malignidade)
     - Categoria 4: Achado suspeito de malignidade:
       - 4A: baixa suspição (10-50%)
       - 4B: suspição intermediária (50-90%)
       - 4C: alta suspição (>90%)
     - Categoria 5: Altamente suspeito de malignidade
     - Categoria 6: Comprovadamente maligno (biópsia positiva)

   ✓ Características de lesões:
     - Forma (oval, redonda, irregular)
     - Margem (circunscrita, micro-lobulada, indistinta, angular, espiculada)
     - Densidade (homogênea, heterogênea, hipoecóica, hiperecóica)
     - Microcalcificações
     - Padrão vascular (ausente, interno, periférico)

   ✓ Tipos comuns:
     - Fibroadenoma (benigno)
     - Fibrocistos (benigno)
     - Papiloma intraductal
     - Carcinoma ductal/lobular

8. RADIOLOGIA ABDOMINAL E PÉLVICA:
   ✓ TC pelve (quando aplicável):
     - Estadiamento de tumores
     - Complicações endometriose (fístulas, obstrução)
     - Patologia colônica associada

   ✓ RM pelve (ressonância):
     - Gold standard para DIE complexa
     - Adenomiose profunda
     - Envolvimento multi-orgão

QUANDO PROCESSAR O DITADO:

1. IDENTIFICAR O CONTEXTO:
   É ginecologia? Obstetrícia? Mastologia? Ou misto?
   O exame é ultrassom? TC? RM? Mamografia?

2. RECONHECER PADRÕES DE LESÃO:
   - Se menciona "cisto com debris" → endometrioma (não cistos simples)
   - Se menciona "halo" → mioma
   - Se menciona "pétalas/lóbulos" → cisto dermóide
   - Se menciona "nódulo hipoecóico infiltrativo" → suspeita de malignidade (O-RADS 4-5)
   - Se menciona "lesão isoecóica peritoneal" → endometriose superficial
   - Se menciona "retração/distorção" → DIE profunda

3. APLICAR CLASSIFICAÇÕES APROPRIADAS:
   Se cisto ovariano → aplicar O-RADS (0-5)
   Se mioma → aplicar FIGO (tipo 0-7)
   Se lesão mamária → aplicar BI-RADS (1-6)
   Se feto → aplicar ISUOG + datação
   Se endometriose → aplicar ENZIAN + compartimentos

4. ESTRUTURAR ACHADOS POR SISTEMA:
   Cada seção (útero, ovário, mama, feto) tem seu próprio JSON com campos específicos

5. ENTENDER MEDIDAS E DIMENSÕES:
   "cisto 4 por 5 centímetros" → 4.0 x 5.0 cm
   "nódulo 2 milímetros" → 0.2 cm ou 2mm
   "massa de 8 centímetros" → 8 cm
   sempre preservar exatamente como ditado + converter para unidade padrão

SEMIOLOGIA ESSENCIAL (O QUE VOCÊ ENTENDE):

PADRÕES DE ECO:
- Anecoico: sem ecos (cisto simples, líquido puro)
- Hipoecóico: menos ecos que estrutura normal (solid, derrame, nódulo)
- Isoecóico: igual à estrutura normal
- Hiperecóico: mais ecos (gordura, calcificação)
- Heterogêneo: múltiplos padrões no mesmo achado

PADRÃO DOPPLER:
- Ausente: sem fluxo (benigno geralmente)
- Periférico: fluxo nas margens (menos sugestivo de malignidade)
- Interno: fluxo dentro da lesão (mais sugestivo de malignidade)
- Globoso: padrão de vasa/hub central (endometriose ativa)

CARACTERÍSTICAS MAIORES DE MALIGNIDADE (O-RADS):
- Papilaridade intracística
- Ascite
- Sólido>2cm
- Parede espessada
- Vascularização central
